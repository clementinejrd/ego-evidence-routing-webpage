# Auxiliary Evidence Routing for Egocentric Video QA

**EPFL CS-503 Visual Intelligence — Spring 2026**

C. Kopidaki, C. Jordan, E. Akcanal, I. Pavlov

---

Project webpage for our CS-503 course project. We study auxiliary evidence selection and question-type-conditioned routing for egocentric video QA, comparing augmentation and replacement modes under a fixed frame budget on the HD-EPIC benchmark.
